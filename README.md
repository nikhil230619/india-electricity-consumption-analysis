# India Electricity Consumption Analysis

## Project Overview
This project is a Power BI dashboard designed to analyze electricity consumption patterns across different states and regions of India. The dashboard provides interactive visualizations and insights to help understand consumption trends, regional distribution, and comparative analysis.

## Features
- State-wise electricity consumption analysis
- Regional consumption comparison
- Interactive charts and filters
- Drill-down analysis for detailed insights
- Data visualization using Power BI

## Tools & Technologies Used
- Power BI
- Data Cleaning
- Data Visualization
- DAX
- Microsoft Excel / CSV Dataset

## Dashboard Insights
- Identifies states with highest electricity consumption
- Compares electricity usage across regions
- Visualizes consumption distribution using charts and maps
- Helps in understanding energy consumption trends

## Visualizations Used
- Clustered Bar Chart
- Stacked Column Chart
- Filled Map / Bubble Map
- KPI Cards
- Slicers and Filters
